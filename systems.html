# OMEGA GOLD — core entry logic
def check_entry(symbol, tf):
# 1. Check session filter
if not is_london_or_ny(): return
# 2. EMA9 crosses above EMA21
ema_cross = ema_cross_up(9, 21)
# 3. ATR confirms volatility
atr_ok = atr(14) > 0.8
# 4. Execute with structured SL/TP
if ema_cross and atr_ok:
place_order(
sl="structure_low",
tp1=1.5R, tp2=2.5R
)

HOW IT THINKS
01
Session FilterLondon or New York session only. No Asian session trades — gold moves with purpose during these windows.
02
15M Regime CheckHigher timeframe must confirm bullish or bearish bias before any 5M entry is considered.
03
EMA Cross EntryEMA9 crossing EMA21 on 5M confirms momentum direction. Cross without ATR confirmation is ignored.
04
ATR Volatility FilterOnly trades when market has real movement. Avoids choppy low-volume periods that kill most strategies.
